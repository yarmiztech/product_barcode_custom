Grouping Product

To create Grouped product from simple products

After successful installation it will show Grouped field inside product form view, by default it will be not checked but if user mark it checked then it will became a grouped product which will contain similar/disimilar products.
After creation of new product, parent product field will be visible to link with it to grouped product, this field is used to create link with the group product which will act as a parent product.
Now in grouped product/ parent product it will display all linked product in Group Ids field which can be similar or disimilar.
